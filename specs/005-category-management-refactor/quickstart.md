# Quickstart: Category Management

## Integration Verification
1. Open the homepage.
2. Verify the FAB now opens a "New Category" interface.
3. Long-press a category card; verify the Action Drawer appears.
4. Verify "Add Task" from the Action Drawer pre-selects the correct category.

## Manual Test Cases
- **Case 1**: Create a category -> Verify it appears in the grid.
- **Case 2**: Long-press Category A -> Edit Name to "Updated A" -> Verify change.
- **Case 3**: Long-press Category A -> Add Task "Test" -> Verify Task appears in Category A details.
