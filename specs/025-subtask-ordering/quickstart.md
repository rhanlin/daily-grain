# Quickstart: SubTask Chronic Ordering

## Verification Steps

### SC-001: Task Detail Ordering
1. Open a Task.
2. Add a subtask "A".
3. Add a subtask "B".
4. Add a subtask "C".
5. Verify order is A -> B -> C.
6. Refresh the page and verify order remains A -> B -> C.

### SC-002: Eisenhower Matrix Sorting
1. Open the Eisenhower Matrix.
2. Filter for "SubTask" or "All".
3. Check a quadrant with multiple subtasks.
4. Verify subtasks are ordered by creation time.

### SC-003: Backlog Consistency
1. Open the Backlog Drawer.
2. Check a category with subtasks from multiple tasks.
3. Verify that all subtasks from the same task are grouped together.
4. Verify that within each group, they are ordered by creation time.
